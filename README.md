A full-stack web application designed to manage SQL database records through complete CRUD (Create, Read, Update, Delete) operations. It provides a clean interface to efficiently handle data using a backend-driven architecture.

---

📌 Features

- ➕ Create new records
- 📖 View all stored data
- ✏️ Update existing records
- ❌ Delete records
- 🔄 Dynamic rendering using EJS
- ⚡ Real-time interaction with MySQL database

---

🛠️ Tech Stack

- Frontend: EJS, HTML, CSS
- Backend: Node.js, Express.js
- Database: MySQL
- Tools & Libraries: Method-Override, UUID / Faker

---

🎯 Project Objective

To build a practical full-stack application that demonstrates how backend systems interact with SQL databases and perform real-world CRUD operations.

---

⚙️ How It Works

The application connects a MySQL database with an Express server. Users can:

- Add new data entries
- View all records from the database
- Edit existing entries
- Delete records when required

EJS templates are used to render dynamic content on the frontend.
